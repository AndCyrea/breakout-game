🧱 Breakout Game (Python Turtle)

A classic **Breakout-style arcade game** built with Python's `turtle` module.  
This project was created as part of my learning journey in game development and programming fundamentals.

---

🎮 Game Overview

The goal is simple:
- Destroy all bricks using the ball
- Keep the ball from falling below the paddle
- Survive with limited lives
- Clear all bricks to win

---

✨ Features

- 🧱 Multiple rows of bricks with different colors
- 🎯 Collision detection (ball ↔ bricks / paddle / walls)
- ❤️ Lives system
- 🏆 Win & Game Over screen
- ⚡ Ball physics with directional movement
- 🎮 Smooth paddle controls
- 🎲 Randomized ball direction after reset

---

🎮 Controls

| Key        | Action            |
|------------|------------------|
| ⬅ Left     | Move paddle left  |
| ➡ Right    | Move paddle right |

---

🧠 What I learned

This project helped me understand:

- Game loops (`while game_is_on`)
- Collision detection using distance checks
- Object movement in 2D space (dx / dy logic)
- State management (lives, win/lose conditions)
- Using lists to manage multiple objects (bricks)
- Basic game architecture
